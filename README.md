# Luisa Score Helper

Userscript yang menambahkan fitur rata-rata nilai, statistik mapel, dan detail penilaian di halaman **Ulangan Harian** [luisa.id](https://www.luisa.id/ulangan_harian).

## Fitur

- **Hitung Rata-rata Keseluruhan** — Rata-rata semua mapel (otomatis mengecualikan Bimbingan Konseling)
- **Statistik Nilai** — Lihat peringkat rata-rata per mapel, tertinggi & terendah
- **Nilai Terbaru** — 15 nilai terbaru dari semua mapel, diurutkan berdasarkan waktu
- **Detail Penilaian** — Klik nama tugas untuk melihat detail (KKM, status lulus, jenis nilai, dll.)
- **Link Cepat** — Klik nama mapel di tabel untuk langsung membuka detail

## Instalasi

1. Install [Violentmonkey](https://violentmonkey.github.io/) atau [Tampermonkey](https://www.tampermonkey.net/)
2. [Klik di sini untuk install userscript](#) *(update link setelah publish di Greasy Fork)*
3. Buka [luisa.id/ulangan_harian](https://www.luisa.id/ulangan_harian)
